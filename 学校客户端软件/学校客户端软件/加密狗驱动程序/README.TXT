           --------------------------------------------------
           软件狗 Windows 9X/ME/NT/2K/XP 驱动安装程序使用说明
           --------------------------------------------------

  软件狗 Windows 9X/ME/NT/2K/XP 驱动安装程序是相应环境下的安装、卸载软件狗
  驱动程序的工具。利用本工具安装的驱动程序可以操作软件狗。本工具可以在
  Windows 9X/ME/NT/2K/XP上运行，安装、卸载相应的驱动程序。

  加密后的 Win32 应用程序在 Windows 9X/ME 下运行时，需要安装软件狗 Windows
  9X/ME的设备驱动程序；在未装驱动的情况下也可正常运行，但在打印时同时操作狗
  可能影响打印。

  加密后的 DOS16，WIN16，WIN32 应用程序在 Windows NT/2K/XP 下运行时，必须要
  安装软件狗 Windows NT/2K/XP 的设备驱动程序。

  运行INSTDRV.EXE点击安装按钮来安装驱动程序，该程序可自动识别操作系统，并提示安装状态。


  如果您需要卸载狗的驱动程序，可以运行INSTDRV.EXE点击卸载按钮来卸载驱动程序，软件可以自
  动识别当前环境，完成驱动程序的卸载工作。

  同时INSTDRV.EXE支持命令行参数，命令行参数（不区分大小写）说明如下：
  /? 显示命令行帮助
  /i 安装时不显示界面
  /r 卸载时不显示卸载界面
  /s 不显示任何安装和卸载信息
  例: instdrv /i/s 安装过程中不显示任何信息
